Официальная документация протокола MoreliaTalk Network¶

Описание протокола актуально на 09.07.2020.

Протокол MoreliaTalk protocol v1 реализован для взаимодействия по типу клиент/сервер в сети MoreliaTalk Network. Интерфейс взаимодействия реализован через ВебСокеты, путём отправки JSON-объекта. Значение первого поля JSON-объекта является имя метода.
